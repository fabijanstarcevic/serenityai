# Serenity AI - Projektni Kod (6. vježba)

Ovaj repozitorij sadrži inicijalni funkcionalni kostur (Proof of Concept) za aplikaciju **Serenity AI**, razvijen prema specifikacijama i personama iz prethodnih vježbi.

## Struktura projekta i raspodjela zadataka:
- `index.html` (Član 3) - Izrada korisničkog sučelja, ekrana za intervenciju i logike tajmera.
- `style.css` (Član 4) - Implementacija vizualnih specifikacija (Dark mode, Fittsov zakon, smirujuća paleta).
- `biometrija_simulacija.js` (Član 2) - Pozadinski AI algoritam za detekciju anomalija pulsa.
- `README.md` (Član 1) - Integracija modula, testiranje sustava i verzioniranje koda.

## Kako pokrenuti simulaciju:
1. Otvorite `index.html` u bilo kojem web pregledniku kako biste vidjeli i testirali sučelje i SOS funkcionalnost.
2. Pokrenite `biometrija_simulacija.js` u Node.js okruženju ili konzoli preglednika kako biste vidjeli rad AI predikcije u realnom vremenu.
